Linux script

Download script: here

Download and run this script on any Linux system that has python 2.7 and crontab installed to create a cronjob automatically that runs periodically.

Input parameters are:

  • period in minutes: cronjob will run every x minutes
  • zone prefix
  • renewal token

For example:

./ddns_updater.sh 5 example abcd1234

will send updates every 5 minutes for the example.ddns.aszabados.eu using token abcd1234
